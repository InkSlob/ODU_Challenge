Judging

A panel of at least 3 judges will evaluate all submissions and rule on the submission that produces the best value to the end user. Judging will be as follows:

  • Creativity - how much “outside the box” thinking was used to:
    • Define the deliverable
    • Implement the solution
  • Usefulness - how valuable and useful is the resulting analysis from the data supplied
  • Efficiency - Speed at which the resulting analysis can be rendered in a demonstrable format.
  • Costs
    • On-going hardware infrastructure costs to support the analytics delivered
    • Ongoing software licensing costs to support the analytics delivered
